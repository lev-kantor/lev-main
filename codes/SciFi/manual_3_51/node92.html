vel_key

Key to either read in atomic velocities from the .<Velocity> box, or thermalise the system initially to dtemp.

Options:

  • 0 - Generate velocities
  • 1 - Read in velocites. Velocities must be provided in the .<Velocity> box (see section 8.5.2)
Default: 0

Example: vel_key 0
