What controls every elementary stage?

Parameters controlling every elementary stage are listed below:

  • amplitude noise ON/OFF (logical);
  • force noise ON/OFF (logical)
  • ADC is ON/OFF (logical):

    • if ON, then the distance is adjusted so that the frequency shift could be maintained aproximately constant close to the set value, (real);
    • if OFF, then the distance is kept exactly constant instead (constant height mode); the frequency will change along the simulation, still kept at resonance;
  • AGC is ON/OFF (logical):

    • if ON, then gain control allows to keep the oscillation amplitude approximately constant close to the set value, (real), by constantly adjusting the excitation amplitude;
    • if OFF, then the excitation amplitude is kept exactly constant, whereas the oscillation amplitude may change along the simulation;
  • velocities and along the and axes, kept constant during every elementary stage (two real);
  • a logical variable which switches ON/OFF the tip-surface force;
  • a logical variable which switches ON/OFF the excitation signal applied to the cantilever;
  • a logical variable which switches ON/OFF the detailed printout during each stage to be made into the file stageM.out (M being the stage number); if ON, then there is also an integer vartiable equal to the number of timesteps after which the printout will be made;
  • finally, presently there are three ways in which each stage can be terminated (a character variable):

    • 'time' followed by the time is seconds;
    • 'freq': the stage will run to meet the frequency shift setpoint for this stage (up to an error of 0.5 Hz);
    • 'ampl': the stage will run to meet the oscillation amplitude setpoint for this stage (up to an error of 0.1 Å).
