do_lateral

If present, the lateral forces on the tip (in the and directions) will also be calculated and dumped in files name.TFX and name.TFY alongside with the force in name.TFZ. As a default, only the force on the tip is calculated. Note that presently, only the -force can be calculated if Image is on, so that the code will stop if do_lateral is chosen together with the Image box.

Options: none

Default: off

Example: do_lateral
