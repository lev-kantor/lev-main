scan

This keyword activates the script-based ``scanning'' mode, when only a move option with a specific displacement (, , ) is added to the final name.RST file after relaxation. This can be used in a shell script to run a series of name.RST files and produce a plot of energy or force vs. distance, with the designated tip atoms moving the specified step in each file.

Options: #

Default: off

Example: scan 0.1 0.1 0.1

Since version 3.11 this option became redundant after introducing the dynamic tip option which does the same job during one job run and thus avoids using complicated shell scripts (see section 8.6). However, it can still be used if desired!
