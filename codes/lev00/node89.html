Working with old academic version of CASTEP

This Chapter describes a rather old implementation of various tools for the CASTEP code. Today this code has become completely redundant and several new versions appeared. Therefore, I decided not to modify this Chapter at all and only have checked for English. Still, some of the routines described here may be of some use with this or other DFT codes, so that I have left this staff here at the end of the manual.
