Cytosine on Au(111): BSSE correction

 
 
 
 
 

1. Start tetr and read in the INPUT_FILE_C_on_Au.fdf input file for SIESTA.

Here we shall obtain the BSSE correction for the adsorption energy of a Cytosine molecule on the Au(111) surface using SIESTA.

2. Once in the main menu, go to the M menu (i.e. type M). Preview the system by invoking W (“write”) and then using xmakemol, see below:

3. We need to split the system into two “molecules”. Make sure that your visualisation tool shows atomic numbers (as above). All gold atoms have numbers 1-96, while Cytosine molecule has numbers 97-109 (use Co to see this!). Then, type B1 with 1-96, below:

4. Now type B1 again with 97-109, see below:

If the division is complete, you will see this in the menu as shown above.

5. Type Co to see how the division was done (the last column):

6. Type B2 to finish the 1st stage. You will be asked if you want the density difference plot to calculate as well. Say ‘n’. The current directory now has a new directory BSSE with subdirectories containing SIESTA input files with “ghost” orbitals:

7. It also contains a simple script BSSE_script.run which should be used for the calculation. Edit the script: you  have to replace the lines calling SIESTA with something you use on your platform (plus other changes if the job is going to be run on a supercomputer). Then run the script.

8. When the job finished, you will find the file BSSE_info containing the desired results of the calculation (these given below were obtained taking the van der Waals interaction into account, do not worry about the numbers, these are shown only for illustration):

The file shows results of 4 calculations needed to work out the BSSE correction given at the very bottom of the table. In this case it was found to be 0.215449 eV.
