Limitations and parameters of the package

The package of computer programs considered in the previous sections has some preset parameters which are scattered over some files with the extension .inc. The meaning of all parameters is described in the corresponding files and do not need to be repeated here. If any of the codes gives an error message stating that some of the parameters are exceeded, go the corresponding *.inc file, change the parameter and recompile the code.
